# YOLOv8 Real-Time Object Detection Pipeline

## Overview
Modular computer vision pipeline for multi-object detection using Ultralytics YOLOv8.

## Project Structure
- `detector.py` → Model loading and inference
- `process_media.py` → Detection pipeline + JSON export
- `visualize.py` → Bounding box annotation
